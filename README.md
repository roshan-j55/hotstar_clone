# 🎬 Hotstar Clone

A full-stack web application replicating the core features and design of the Hotstar streaming platform. Built using **Django** and **Python**.

## 🚀 Features
- **Video Playback:** Seamless media streaming for various video formats.
- **Dynamic Content:** Media catalog driven by a SQLite database.
- **Modern UI:** Responsive template design mimicking a premium streaming service.

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Database:** SQLite3
- **Frontend:** HTML, CSS (Django Templates)

## 💻 Running the Project Locally

Follow these steps to get your development environment set up:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/roshan-j55/hotstar_clone.git
   cd hotstar_clone
   ```

2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use: venv\Scripts\activate
   ```

3. **Install dependencies:**
   *(Ensure you have Django installed in your environment)*
   ```bash
   pip install django
   ```

4. **Apply database migrations:**
   ```bash
   python manage.py migrate
   ```

5. **Run the local development server:**
   ```bash
   python manage.py runserver
   ```

6. Open your web browser and go to `http://127.0.0.1:8000` to see the app running!
